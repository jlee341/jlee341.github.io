SnailBot Privacy Policy


Applications

This privacy policy applies to the following web applications:


  • SnailBot#3024 - Public Bot
  • SnailBot#5062 - Non-Public Testing Bot
  • SnailBot#2571 - Non-Public Testing Bot
  • SnailBot#9974 - Non-Public Testing Bot

This privacy policy applies to the following websites:

  • Snailbot Official Website / Documentaion - https://snailbot.net

Terminology


Server Administrator

  • Any user who has the permission, ability, or access to add/configure Snailbot to a server.

Server Member

  • Any user who is a member of a server which SnailBot is present on.

SnailBot Team Member

  • Anyone who is authorized to access Snailbot dev/staff tools or websites

Data Collected By Command


The following may be collected via user run command, and may be enterd by the user, voluntarily. This data is not collected automatically. When providing data in this way, you forefo any rights to the content of the data provided.


  • Server configurations (region, name, id)
  • Server Ownership
  • List of server's members
  • List of server's roles
  • Server and Snailbot Partnership status

Data Collected When Features Are Enabled


the following will be collected automatically when a server administrator configures the bot to perform checks for news or another automated feature with SnailBot. These features are not enabled automatically, and therefore the following will not be collected automatically unless the corresponding feature is enabled by the server administrator.


  • Server configurations (channel names, channel ids)

  • List of server's roles

  • List of server's member's

Data Collected Automatically


The following is collected automatically when Snailbot joins a server. This data is used to provide statistics or server information.


  • Server ownership

  • Server member count

  • Server channels

  • Server availability

  • List of roles

  • Server & Snailbot Partnership status

  • Message Content (not queried unless Message Edit Logging and Message Delete Loggiing are enabled)

Data Collected via Authorization


The following is collected when creating an account on the SnailBot Documentation website ( https://snailbot.net). This information is explicitly specified by the user, and collected by us when the user enters the data:


  • Email address
  • Password
  • Desired Username

Data Storage & Protection


All data is stored on secured servers. Maximum efforts are taken to keep collected data, as defined above, protected.

SnailBot will store message content in memory only, and will only be used by the moderation tools if they are enabled


Server Member Agreement

By being a member of a server which uses SnailBots' services or features, you are consenting to the policies outlined in this agreement. If you, the server member, do not agree with any policies outlined in this agreement, you have the right to leave the server in which this bot is a part of.


Server Administrator Agreement

By adding SnailBot to your server, you are consenting to the policies outlined in this agreement. If you, the server administrator, do not agree with any of the policies outlined in this agreement, you have the right to remove SnailBot from your server.


SnailBot Team Member Agreement

By being a member of the SnailBot Team and holding access to back-end development and administrative tools for SnailBot, you are consenting to the policies outlined in this agreement. If you, the SnailBot Team Member, do not agree with any policies outlined in this agreement, you have the right to ask for removal of your access to the services.
