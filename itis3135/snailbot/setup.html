Automatic Feed Setup

This will guide you through how to setup automated news posting in SnailBot.


Note:Automated Feed Setup has drastically changed since V.2.2, taking advantage of buttons, select menus and forms to help set up a news feed.

Disabling and Listing Active Automatic Feeds


To disable a feed, use /disablefeed. Discord will autocomplete providers and enabled feeds. To see which feeds are enabled, use /listfeed.
